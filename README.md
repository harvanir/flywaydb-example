# flywaydb example (MySQL)

**1. Prerequisite**

* Docker
* Java 8+

**2. Run docker compose**

Command:

```bash
./run-db
```

**3. Run maven for the first time**

Command:

```bash
./mvnw clean compile
```

**4. Run maven for the first time**

Command:


